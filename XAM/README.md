# Мессенджер ХАМ

Простой мессенджер с регистрацией по номеру телефона.

## Особенности

- Регистрация по номеру телефона (формат: 7XXXXXXXXXX или 8XXXXXXXXXX)
- Пароль не менее 5 символов
- Ограничение: 3 аккаунта с одного IP-адреса
- Личные и групповые чаты
- Темная и светлая темы
- Адаптивный дизайн для ПК и мобильных устройств

## Установка

Просто загрузите файлы на любой веб-хостинг.

## Использование

1. Зарегистрируйтесь с номером телефона и паролем
2. Найдите пользователей для начала чата
3. Общайтесь в личных и групповых чатах
4. Настройте тему в настройках профиля
